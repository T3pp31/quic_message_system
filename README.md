# QUIC接続をRustで実装してみる

Quic接続をRustで実装し，メッセージを送れるプログラムを作成し，Quicに対する理解を深める
quic接続を理解できるようになったら，簡単に接続できる関数を作成し，Cargo.ioにPublishする
